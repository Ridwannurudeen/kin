# Changelog

## 0.1.0 - initial release

- Extracted Hunt's attestation digest primitive into a standalone SDK.
- Added canonical class registries for smart-contract CWEs, insurance defects, benefits-defense defects, and medical reading classes.
- Added ECIES helpers for secp256k1 wallet public keys.
- Added attestation signing and verification wrappers for ethers signers.
- Added runnable examples for all four Hunt verticals plus a generic classification domain.
